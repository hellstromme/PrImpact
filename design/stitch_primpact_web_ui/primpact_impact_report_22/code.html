Async over Sync

Enforcing Promise-based storage prevents UI blocking on heavy write operations. Assumes environment supports ES2020.

Rationale

"System observability requires non-blocking I/O to maintain 60fps."

D-02
warning MED RISK

Adapter Pattern

Using an Adapter to shim the old interface. Risk of memory leak if listeners aren't properly detached.

Rationale

"Ensures no breaking changes for the web client during migration phase."

D-03
shield LOW RISK

JSON Serialization

Assuming all persisted objects are serializable. Private class fields will be dropped by the new serializer.

Rationale

"Standardizes the wire format for future multi-platform sync."
